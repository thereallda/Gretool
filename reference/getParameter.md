# Accessor of parameters

List all parameters.

Get specific parameters.

## Usage

``` r
listParameter(object)

getParameter(object, name)

# S4 method for class 'Gretool'
listParameter(object)

# S4 method for class 'Gretool,character'
getParameter(object, name)
```

## Arguments

- object:

  Gretool.

- name:

  Name of the parameter.

## Value

Vector of parameter names

Vector of parameter
