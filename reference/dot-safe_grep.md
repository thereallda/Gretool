# Safe grep

Safe grep

## Usage

``` r
.safe_grep(pattern, x, ...)
```

## Arguments

- pattern:

  Same as `grep`. character string containing a regular expression (or
  character string for fixed = TRUE) to be matched in the given
  character vector.

- x:

  Same as `grep`. a character vector where matches are sought, or an
  object which can be coerced by as.character to a character vector.

- ...:

  Other parameters passed to `grep`

## Value

matched value
